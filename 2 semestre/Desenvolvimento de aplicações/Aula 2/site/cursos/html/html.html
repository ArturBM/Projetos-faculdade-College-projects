Curso de HTML

Conteudo HTML

  1. Introdução ao HTML: Explicação sobre o que é HTML, sua estrutura básica, e como ele funciona para criar a estrutura de uma página web.

  2. Elementos e Tags: Detalhes sobre as diferentes tags HTML, como h1 a h6 para títulos, p para parágrafos, a para links, img para imagens, e muitas outras que formam a base do conteúdo de uma página.

  3. Atributos HTML: Como utilizar atributos para adicionar informações adicionais aos elementos, como src para imagens, href para links, alt para texto alternativo, e class ou id para estilização com CSS.

  4. Formulários: Criação de formulários interativos utilizando tags como input, textarea, button, e select, permitindo a recolha de dados dos utilizadores.

  5. Tabelas e Listas: Como organizar e apresentar dados usando tabelas (table, tr, td) e listas ordenadas e não ordenadas (ul, ol, li).

  6. Estrutura de Página: Princípios de layout básico, como dividir a página em cabeçalho (header), rodapé (footer), navegação (nav), e secções de conteúdo (section, article).

  7. Acessibilidade e SEO: Melhores práticas para tornar as páginas acessíveis a todos os utilizadores e otimizar o HTML para motores de busca.
